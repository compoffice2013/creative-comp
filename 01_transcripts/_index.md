# 🎙️ 文字起こしアーカイブ

> イベント、講演、ラジオ、会議の文字起こし原文を保管するフォルダです。
> ブログ記事になる前の「素材」をここに蓄積します。

## フォルダ分類ガイド

| フォルダ | 保管する内容 | 例 |
|---------|------------|-----|
| `events/` | パーティ、イベント、お祝いの場の文字起こし | 新年会、結婚式、記念式典 |
| `lectures/` | 講演、セミナー、勉強会の原稿・文字起こし | 白山商工会講演、倫理法人会講話 |
| `radio/` | ラジオ出演、インタビューの文字起こし | しゃべっちゃお放送回 |
| `meetings/` | 会議、打ち合わせ、相談の議事録 | 県三役会、委員会ミーティング |

## ファイル命名規則

```
YYYY-MM-DD-キーワード.txt  （テキスト原文の場合）
YYYY-MM-DD-キーワード.md   （Markdown整形済みの場合）
```

## 文字起こし → ブログ の流れ

```
1. 文字起こし原文を取得
   → 01_transcripts/ の適切なサブフォルダに保存

2. ブログ記事を生成（/blog-writer ワークフロー）
   → 02_blog/drafts/ に保存

3. 講演録として保管する場合
   → 04_knowledge/rinri/ に保存（倫理関連）
   → 04_knowledge/ 直下に保存（その他講演）
```

## 注意事項

- 文字起こし原文には本名が含まれる場合がありますが、**保管はOK**です
- ブログ化する際に本名が出力されないよう、ワークフローで制御しています
