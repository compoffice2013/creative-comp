# 📝 プレースホルダファイル

> このフォルダには、過去のWordPressブログからインポートした際に作成されたものの、
> 中身がまだ取得できていない空ファイルが格納されています。

## 対応方法

1. WordPressの管理画面から該当記事の本文を取得する
2. 取得したら親フォルダ（`rinri/`）に移動し、YAML フロントマターを追加する
3. 空のままであれば、このフォルダに残しておく

## フォーマット（移動時に付与）

```markdown
---
title: "タイトル"
date: YYYY-MM-DD
url: https://comp-office.com/blog/XXXX/
---

# タイトル

本文...
```
