# 📝 ブログ執筆ワークフロー

このワークフローは、音声から文字起こしされたテキストを元に、コンプレッサー様の文体でブログ記事、SEO設定、Facebook投稿を生成します。

## 手順

1. **インプットの準備**
   - `d:\blog\transcripts\` 内に新しいテキストファイルを作成し、文字起こしデータを貼り付けてください。

2. **解析と生成**
   - 以下の内容を `persona.md` のルールに従って生成します。
   - **ブログ本編**: H2/H3見出しを活用したSEO最適化済み記事。一人称を一切使わない。
   - **SEO項目**: 
     - フォーカスキーワード
     - SEOタイトル（クリックしたくなるもの）
     - メタディスクリプション
   - **Facebook用投稿**: 現場の熱量が伝わるSNS形式。
   - **アイキャッチ画像案**: DALL-E等で生成するためのプロンプト案。

3. **ファイルの保存**
   - 生成物は `d:\blog\drafts\` 内に日付付きで保存します。

4. **Gitへの保存**
   - 保存完了後、自動的にGitにコミットします。

## 実行方法
私（Antigravity）に、「〇〇（ファイル名）を使ってブログを書いて」と指示してください。
